## 🎯 如何使用

### 方式 1️⃣：使用 DMG 文件（推荐）
1. 下载 `成绩计算器-0.0.0-arm64.dmg`
2. 双击打开DMG文件
3. 将"成绩计算器"图标拖到"应用程序"文件夹
4. 打开"应用程序"文件夹
5. 双击"成绩计算器"即可运行

### 方式 2️⃣：使用 ZIP 文件
1. 下载 `成绩计算器-0.0.0-arm64-mac.zip`
2. 解压缩
3. 将"成绩计算器.app"复制到"应用程序"文件夹
4. 双击"成绩计算器.app"即可运行
